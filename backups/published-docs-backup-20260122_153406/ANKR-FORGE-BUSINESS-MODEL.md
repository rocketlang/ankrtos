# AnkrForge - Business Model

> Revenue streams, pricing strategy, and go-to-market plan

---

## Business Model Overview

AnkrForge operates as a **platform business** with multiple revenue streams, combining elements of marketplace, SaaS, and manufacturing-as-a-service.

```
┌─────────────────────────────────────────────────────────────────────────┐
│                       ANKRFORGE BUSINESS MODEL                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│    CONSUMERS ←──────→ ANKRFORGE PLATFORM ←──────→ MANUFACTURERS        │
│         │                     │                         │               │
│         │                     │                         │               │
│    Buy custom           Platform takes            Receive orders        │
│    products             commission                fulfill & ship        │
│                               │                                         │
│                               ↓                                         │
│                        MODULE DEVELOPERS                                │
│                               │                                         │
│                        Build product                                    │
│                        modules, earn                                    │
│                        revenue share                                    │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘
```

## Revenue Streams

### 1. Transaction Fees (Primary)

Commission on each product sold through the platform.

| Component | Rate | Example (₹5000 product) |
|-----------|------|------------------------|
| **Platform Fee** | 15% | ₹750 |
| **Payment Processing** | 2% | ₹100 |
| **Total Take Rate** | 17% | ₹850 |

**Revenue to Stakeholders:**
- Module Developer: 70% of product price = ₹3,500
- Manufacturing Partner: Cost + margin = ₹600
- AnkrForge: 15% platform fee = ₹750
- Payment Gateway: 2% = ₹100
- Shipping: Pass-through = ₹50 (paid by customer)

### 2. Subscription Tiers (B2C)

Optional premium features for consumers.

| Tier | Price | Features |
|------|-------|----------|
| **Free** | ₹0 | 3 scans/month, standard shipping |
| **Forge+** | ₹299/month | Unlimited scans, priority manufacturing, free shipping on orders >₹2000 |
| **Forge Pro** | ₹999/month | All Forge+, early access to modules, exclusive products, dedicated support |

**Projected Distribution:**
- Free: 80% of users
- Forge+: 15% of users
- Forge Pro: 5% of users

### 3. Partner/Enterprise (B2B)

White-label and API access for businesses.

| Plan | Price | Features |
|------|-------|----------|
| **Starter** | ₹25,000/month | 500 API calls, 2 modules, basic support |
| **Business** | ₹75,000/month | 5,000 API calls, unlimited modules, priority support, white-label |
| **Enterprise** | Custom | Unlimited, dedicated infrastructure, custom modules, SLA |

**Target Customers:**
- Audio brands wanting custom-fit earbuds
- Footwear companies adding custom insoles
- Healthcare providers for orthotics
- Sports brands for protective gear

### 4. Module Marketplace

Revenue from third-party module developers.

| Model | Structure |
|-------|-----------|
| **Revenue Share** | 30% of module sales to platform |
| **Featured Placement** | ₹50,000/month for homepage feature |
| **Certification** | ₹25,000 one-time for "Forge Certified" badge |

### 5. Manufacturing Network Fees

Fees from manufacturing partners.

| Fee Type | Rate |
|----------|------|
| **Onboarding** | ₹50,000 one-time |
| **Transaction Fee** | 5% of manufacturing value |
| **Priority Routing** | ₹25,000/month for priority job routing |

### 6. Data & Insights (Future)

Anonymized body measurement data and trends.

| Product | Price |
|---------|-------|
| **Industry Reports** | ₹2,00,000/report |
| **Custom Research** | ₹5,00,000+ |
| **API Access** | ₹50,000/month |

## Pricing Strategy

### Consumer Product Pricing

```
Final Price = Base Cost + Customization Premium + Platform Margin + Shipping

Where:
- Base Cost = Manufacturing + Materials + Module Developer Fee
- Customization Premium = Complexity-based markup
- Platform Margin = 15-20%
- Shipping = Distance-based
```

### Example: Custom Earbuds

| Component | Amount (₹) |
|-----------|------------|
| Manufacturing Cost | 800 |
| Material Cost | 400 |
| Module Developer (70% of margin) | 2,100 |
| Platform Fee (15%) | 750 |
| Payment Processing (2%) | 100 |
| **Subtotal** | **4,150** |
| Customization (premium color) | 500 |
| Shipping | 99 |
| **Total to Customer** | **₹4,749** |

### Competitive Pricing Analysis

| Product | Market Price | AnkrForge Price | Savings |
|---------|--------------|-----------------|---------|
| Custom Earbuds | ₹8,000-15,000 | ₹4,500-6,000 | 40-60% |
| Custom Insoles | ₹3,000-8,000 | ₹2,000-4,000 | 33-50% |
| Custom Mouthguard | ₹5,000-12,000 | ₹2,500-5,000 | 50-60% |

**Competitive Advantage**: Shared infrastructure reduces costs across all verticals.

## Go-To-Market Strategy

### Phase 1: Launch (Months 1-6)

**Focus**: Single vertical, prove the model

#### Target Vertical: Custom Earbuds (ForgeAudio)
- Large market (₹12,000 Cr in India)
- High willingness to pay for customization
- Clear pain point (poor fit)
- Tech-savvy early adopters

#### Launch Strategy
1. **Pre-launch Waitlist**: Build 10,000+ waitlist
2. **Influencer Seeding**: 50 tech/audio influencers
3. **Launch Offer**: First 1,000 customers at 40% off
4. **PR**: Tech media coverage (YourStory, Inc42, TechCrunch India)

#### Channels
| Channel | Budget | Target |
|---------|--------|--------|
| Instagram/YouTube Influencers | ₹10L | 50K signups |
| Google Ads | ₹5L | 20K signups |
| Tech PR | ₹2L | Media coverage |
| Community Building | ₹3L | Discord/WhatsApp groups |

### Phase 2: Expansion (Months 7-12)

**Focus**: Add verticals, expand user base

#### New Verticals
- **ForgeFoot**: Custom insoles (partner with running communities)
- **ForgeDental**: Mouthguards (partner with gyms, sports clubs)

#### Partnerships
- Audio brands for co-branded products
- Fitness chains for in-store scanning
- Corporate wellness programs

#### Geographic Expansion
- Tier 1 cities: Mumbai, Delhi, Bangalore, Chennai, Hyderabad
- International: UAE, Singapore (pilot)

### Phase 3: Platform (Months 13-24)

**Focus**: Open platform, B2B growth

#### Module Marketplace Launch
- Open SDK to third-party developers
- Developer incentive program (₹1Cr fund)
- Hackathons and module challenges

#### Enterprise Push
- Dedicated sales team
- Industry-specific solutions
- White-label partnerships

## Competitive Moat

### Short-Term Moats
1. **First-Mover**: First platform approach in India
2. **Technology**: Proprietary scan processing AI
3. **Manufacturing Network**: Exclusive partner relationships

### Long-Term Moats
1. **Network Effects**: More modules → more users → more manufacturers → lower costs
2. **Data Moat**: Largest database of 3D body scans
3. **Brand**: "Forged for you" becomes synonymous with custom-fit
4. **Ecosystem**: Developer community building modules

## Risk Mitigation

| Risk | Mitigation |
|------|------------|
| **Low initial adoption** | Focus on one vertical, prove value |
| **Manufacturing quality** | Strict QA, money-back guarantee |
| **Competition from giants** | Move fast, own the niche |
| **Technology challenges** | Hire top AI talent, partner with labs |
| **Regulatory (medical devices)** | Start with non-medical products |

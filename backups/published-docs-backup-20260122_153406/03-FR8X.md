## AI & Agent Orchestration

### Multi-Agent System
- **5 LLM Providers:** Claude Haiku/Sonnet/Opus, GPT-4o, GPT-3.5
- **Agent Pools:** FAST_POOL, BALANCED_POOL, QUALITY_POOL, COST_OPTIMIZED_POOL
- **Task Classification:** Simple/Medium/Complex; code/docs/architecture/logistics
- **Competition-based Selection:** Agents compete, winners evaluated on quality/speed/cost

### 10 Bot Types (All with 10-language support)
1. **DRIVER_BOT** - Navigation, loads, compliance alerts
2. **CARRIER_BOT** - Fleet management, settlements
3. **SHIPPER_BOT** - Rate queries, tracking
4. **WAREHOUSE_BOT** - Slot optimization, inventory
5. **LABOR_BOT** - Job matching, payments
6. **CUSTOMS_BOT** - HS codes, documentation
7. **INSURANCE_BOT** - Claims, policies
8. **WALLET_BOT** - Balance, payments, settlements
9. **SUPPORT_BOT** - Help, escalations
10. **ONBOARDING_BOT** - Verification, guidance

### Swayam Universal Bot
- Hindi + 9 Indian languages
- Text, voice, mixed modes
- 8 personas (Shipper, Carrier, Driver, Broker, Compliance, Pricing, Operations, Support)

### Memory Integration (@ankr/eon)
- Episodic memory (interactions, history)
- Semantic memory (knowledge graphs)
- Procedural memory (workflows)
- Contextual awareness (preferences)

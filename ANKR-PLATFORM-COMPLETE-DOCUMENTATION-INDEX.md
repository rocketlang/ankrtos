### **RAG (Retrieval Augmented Generation)**

**What it is:**
A smart way to make AI answer questions from YOUR documents (not random internet).

**How it works:**
```
1. Upload textbook â†’ Extract text
2. Break into chunks (512 words each)
3. Create "embeddings" (mathematical fingerprints)
4. Store in vector database (pgvector)
5. When student asks question:
   â†’ Search for relevant chunks
   â†’ Send to AI to write answer
   â†’ Show with page numbers
```

**Why it's better than fine-tuning:**
- âœ… Cheaper (use existing AI models)
- âœ… Faster (works instantly)
- âœ… Updatable (just upload new docs)
- âœ… No ML expertise needed

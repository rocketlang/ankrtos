## 🛠️ Technical Implementation

### Tech Stack

```yaml
Frontend:
  Framework: React 19 + TypeScript 5.9
  Build: Vite 6.x
  Styling: TailwindCSS 4.x + Shadcn/UI
  State: Zustand + Apollo Client
  Charts: Recharts + D3.js (for memory graph)
  3D: Three.js / React Three Fiber (memory visualization)
  Voice: Web Speech API + Sarvam TTS
  Animation: Framer Motion

Backend Integration:
  GraphQL: Apollo Server (existing endpoints)
  WebSocket: Socket.io (real-time updates)
  MCP: Direct tool invocation
  Voice: Swayam WebSocket (7777)

Deployment:
  Hosting: Vercel / Cloudflare Pages
  CDN: Cloudflare
  Analytics: PostHog
  Error Tracking: Sentry
```

TROUBLESHOOTING
---------------------------------------
If you get "Permission denied":
- Your SSH key is already set up on server
- Need to add same key to phone:
  cat ~/.ssh/id_ed25519.pub
  (Add to GitHub if different device)

If repos are in different folder:
- Find them: find ~ -name ".git" -type d
- Move them: mv /old/path ~/projects/

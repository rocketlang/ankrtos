# OpenCode IDE User Guide

## Getting Started

### 1. Create a Session

1. Click the **Sessions** tab
2. Click **+ New Session**
3. Enter project name and directory
4. Click **Create**

### 2. Explore Tools

1. Go to **Tools** tab
2. Browse 703 available tools
3. Use search or filters to find tools
4. Click on a tool to see details

### 3. Execute Tools

1. Select a tool
2. Fill in the required parameters
3. Click **Execute**
4. View results in **Executions** tab

## Features

### Code Editor

- **Multi-tab editing** - Open multiple files
- **Syntax highlighting** - 20+ languages
- **Auto-save** - Optional automatic saving
- **Format** - Format document (Cmd+Shift+F)

### AI Assistant

#### Generate Code
1. Go to **AI** tab
2. Select **Generate Code**
3. Choose language
4. Describe what you want
5. Click **Generate**

#### Code Review
1. Open a file in editor
2. Go to **AI** tab
3. Select **Review Code**
4. Click **Execute**
5. View issues and score

### Voice Controls

1. Go to **Voice** tab
2. Click **Start** to begin listening
3. Speak your command
4. View transcript in real-time

**Commands:**
- "Execute [tool name]"
- "Search [query]"
- "Open [file]"

### Session Memory

1. Go to **Memory** tab
2. Click **+ New Memory**
3. Select type:
   - **Semantic** - Facts & knowledge
   - **Episodic** - Events & experiences
   - **Procedural** - How-to & skills
   - **Preference** - User preferences
4. Enter content
5. Set importance (0-1)
6. Click **Store**

### Git Integration

1. Open **Git** panel
2. **Stage files** - Click Stage on changed files
3. **Commit** - Enter message and click Commit
4. **Push** - Click Push to sync with remote
5. **Branch** - Create/switch branches

## Keyboard Shortcuts

- `Cmd+1-5` - Switch tabs
- `Cmd+K` - Command palette
- `Cmd+S` - Save file
- `Shift+?` - Show shortcuts

## Settings

Access via **Settings** tab:

- **Theme** - Light/Dark/System
- **Editor** - Font size, tab size, word wrap
- **Budget** - Daily/monthly limits
- **Notifications** - Sound, desktop alerts

## Tips

1. **Use Favorites** - Star frequently used tools
2. **Filter by Session** - Track tool usage per session
3. **Export Data** - Download execution logs as CSV/JSON
4. **Split View** - View two panels side-by-side
5. **Important Memories** - High importance for critical info
